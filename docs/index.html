しごとらアプリについて

株式会社オープンループパートナーズが運営する日本最大級の日払い・未経験者歓迎の求人情報検索サイト「しごとら」のスマホ用アプリ。1日のみ・1週間以内、短期・単発・長期、日払い・週払い・月払い、アルバイト・派遣・社員・転職や、オープニングスタッフ、時給1,400円以上の時給の仕事、未経験者・高校生・大学生歓迎のお仕事、コールセンター・テクニカル・官公庁関連・事務・データ入力・封入・試験監督・軽作業・製造・ライン・フード・接客・キッティングなどの人気の職種！！毎日募集情報が更新されるので、最新の求人情報が見つかります！

毎日が給料日

しごとらは、ご紹介しているお仕事のほとんどは「給与全額日払い」が可能なお仕事です。
しかも、銀行振込手数料も無料で、働いた初日分から支払い可能。
給料を受け取りに行くなんて、もう古い！必要書類の事前FAXなど面倒な手続きも不要なので、働いた翌日から毎日がお給料日になります。

充実の特集

即就業可能な寮付き・寮完備の県外求人特集や、40代以上の方の豊富な実務経験を活かせる特集、医療・介護系の求人特集から、正社員での転職を目指す方向けの転職エージェントサービスの特集など、様々な特集を定期的に更新しています。
また、大人気のコールセンターや販売職、製造職などの人気職種の特集コンテンツもご用意。
